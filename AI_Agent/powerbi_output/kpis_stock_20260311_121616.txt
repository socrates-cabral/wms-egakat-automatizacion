## 01 — MEDIDAS BASE

```dax
-- ============================================================
-- Stock_Total_Pallets
-- Cuenta el total de pallets registrados en WMS
-- Incluye todos los estados (bloqueados y disponibles)
-- ============================================================
Stock_Total_Pallets =
VAR _total =
    CALCULATE(
        DISTINCTCOUNT( Stock_WMS[Pallet] ),
        ALL( Stock_WMS[Bloqueado] )   -- ignora filtro de bloqueo para total real
    )
RETURN
    IF( ISBLANK( _total ), 0, _total )

-- Formato : #,##0
-- Alerta   : Sin umbral (métrica base)
```


## 02 — MEDIDAS BLOQUEADOS

```dax
-- ============================================================
-- Stock_Pallets_Bloqueados
-- Pallets con flag Bloqueado = "S" (Sí) en WMS
-- Origen típico: cuarentena, vencimiento, daño físico
-- ============================================================
Stock_Pallets_Bloqueados =
VAR _bloqueados =
    CALCULATE(
        DISTINCTCOUNT( Stock_WMS[Pallet] ),
        Stock_WMS[Bloqueado] = "S"
    )
RETURN
    IF( ISBLANK( _bloqueados ), 0, _bloqueados )

-- Formato : #,##0
-- Alerta   : Verde < 5% | Amarillo 5–10% | Rojo > 10%
```

```dax
-- ============================================================
-- Stock_Bultos_Bloqueados
-- Bultos totales asociados a pallets bloqueados
-- ============================================================
Stock_Bultos_Bloqueados =
VAR _bultos_blq =
    CALCULATE(
        SUM( Stock_WMS[Bultos] ),
        Stock_WMS[Bloqueado] = "S"
    )
RETURN
    IF( ISBLANK( _bultos_blq ), 0, _bultos_blq )

-- Formato : #,##0
-- Alerta   : Ídem % Bloqueado
```

```dax
-- ============================================================
-- Stock_Pct_Bloqueado_Pallets
-- Porcentaje de pallets bloqueados sobre total
-- KPI PRINCIPAL de calidad de inventario
-- Fórmula: Bloqueados / Total * 100
-- ============================================================
Stock_Pct_Bloqueado_Pallets =
VAR _bloqueados  = [Stock_Pallets_Bloqueados]
VAR _total       = [Stock_Total_Pallets]
VAR _pct         = DIVIDE( _bloqueados, _total ) * 100
RETURN
    ROUND( _pct, 2 )

-- Formato : 0.00"%"
-- Umbral   : 🟢 < 5%  |  🟡 5% – 10%  |  🔴 > 10%
```

```dax
-- ============================================================
-- Stock_Pct_Bloqueado_Bultos
-- Porcentaje de bultos bloqueados sobre total de bultos
-- Complementa el KPI por pallet (pallets pequeños vs grandes)
-- ============================================================
Stock_Pct_Bloqueado_Bultos =
VAR _blq   = [Stock_Bultos_Bloqueados]
VAR _total = [Stock_Total_Bultos]
RETURN
    ROUND( DIVIDE( _blq, _total ) * 100, 2 )

-- Formato : 0.00"%"
-- Umbral   : 🟢 < 5%  |  🟡 5% – 10%  |  🔴 > 10%
```

```dax
-- ============================================================
-- Stock_Bloqueado_Por_Estado
-- Suma de pallets bloqueados agrupados por Descripcion_Estado
-- Permite ver si el bloqueo es por cuarentena, vencimiento, etc.
-- Diseñada para usar en visual de tabla/matriz con fila Estado
-- ============================================================
Stock_Bloqueado_Por_Estado =
CALCULATE(
    DISTINCTCOUNT( Stock_WMS[Pallet] ),
    Stock_WMS[Bloqueado] = "S"
    -- El filtro de Descripcion_Estado viene del contexto del visual
)

-- Formato : #,##0
-- Alerta   : Sin umbral directo (diagnóstico)
```

```dax
-- ============================================================
-- Stock_Alerta_Bloqueado_Flag
-- Devuelve texto de semáforo para uso en tarjetas KPI
-- 🟢 Normal | 🟡 Atención | 🔴 Crítico
-- ============================================================
Stock_Alerta_Bloqueado_Flag =
VAR _pct = [Stock_Pct_Bloqueado_Pallets]
RETURN
    SWITCH(
        TRUE(),
        _pct < 5,                "🟢 Normal",
        _pct >= 5 && _pct < 10,  "🟡 Atención",
        _pct >= 10,              "🔴 Crítico",
        "⚪ Sin datos"
    )

-- Formato : Texto
-- Uso     : Tarjeta de estado en dashboard ejecutivo
```
